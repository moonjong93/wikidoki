# wikidoki
누구나 무료로 사용할 수 있는 위키 시스템

## 무엇을 만드나요?
위키 시스템을 만듭니다. 위키란 [나무위키](https://namu.wiki/w/%EB%82%98%EB%AC%B4%EC%9C%84%ED%82%A4:%EB%8C%80%EB%AC%B8)와 같은 '누구나' 비영리적인 목적으로 문서를 만들어 '집단 지성'의 문서들을 생성해내는 것을 말합니다.

## 스텍
1. Node.js
1. Vue.js (Juqery)
1. CSS
1. HTML
